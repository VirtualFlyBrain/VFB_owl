## JSON files for easy export of VFB individuals data

JSON structure: 

~~~.javascript

{ shortFormID_string : 
      { 'label': string,
        'def': string,
        'types': [ 
         { 'isAnonymous': boolean, 
           'relId': shortFormID_string,
           'objectId': shortFormID_string
          },
          { ... },
          ]
      }
  }

~~~~~

e.g.

~~~.javascript

{
    "VFB_00020095": {
        "Types": [
            {
                "isAnonymous": false, 
                "objectId": "FBbt_00007683"
            }, 
            {
                "isAnonymous": false, 
                "objectId": "FBbt_00110357"
            }, 
            {
                "isAnonymous": true, 
                "objectId": "FBbt_00003624", 
                "relId": "BFO_0000050"
            }
        ], 
        "def": "An example of an adult EBp1 lineage clone in the adult brainThis clone was generated by FLP-out MARCM at larval hatching. It therefore does not include embryonic born neurons.", 
        "general_classification": "FBbt_00007683", 
        "label": "EBp1 clone of Ito 2013"
    },
    
~~~~~
